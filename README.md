# HOMIFY
## WHAT YOUR PRODUCT DOES: 
This is in application that allows the buying an selling of home related products,materials and professional services which includes home decor, materials used for interior architechural modificstion/designs.
most materials used in home building and modifications are usually gotten fron local artisans

## WHY IT MATTERS: 
In situations where buyers are sometimes given the wrong order or receive poor quality goods, that’s where we step in. Our application has a verification process for artisans, ensuring that only verified artisans can sell materials. It also contains an escrow system that allows us to withhold payment once it’s made, ensuring customer satisfaction with the goods. A picture segment is included for evidence, and a satisfaction button on both sides (both vendor and buyer) is used to complete the transaction. Once payment is made and the customer is satisfied, the payment is credited to the vendor.

## This application is for:
Interior designers that work on the architectural aspect.
Companies selling goods, e.g., chairs, wood, rubber pipes, taps.
Entrepreneurs selling second-hand or first hand goods, e.g., furniture, home décor items, lighting materials.
Repair professionals and home renovators.

## FEATURES
AN ESCROW SYSTEM: Secure payment holding to ensure safe transactions between buyers and sellers.
SERVICE LISTINGS: Professionals can list their services with details, pricing, and images.
SEARCH BAR: Users can quickly find services, products, or professionals.
FILTER OPTIONS: Narrow down results by category, style, location, or price.
CATEGORIES&TAG: Organizes services and products for easy discovery.
PROFESSIONAL LISTING: Detailed profiles showcasing expertise, portfolio, and credentials.
INSPIRATION BOARD: Users can save ideas and project examples for future reference.
DIRECT COMMUNICATION: Buyers can contact sellers or request quotes directly.


## TECHNOLOGIES:
Escrow System : Payment gateways (Stripe, PayPal), backend logic (Node.js, Python, Ruby)
Service Listing: Backend frameworks (Django, Rails, Node.js), database (PostgreSQL, MongoDB)
Search Bar: Elasticsearch, Algolia, or built-in database search
Filter Section: Frontend frameworks (React, Vue.js), API endpoints with query filtering
Category & Tag System: Relational database (PostgreSQL, MySQL), backend APIs
Professional Listing: CMS or custom backend, cloud storage for portfolios (AWS S3, Firebase)
Inspiration Board: Frontend frameworks, cloud storage for images, user session management
Direct Communication: Messaging APIs (Twilio, Firebase Realtime DB), WebSockets
